# PocketClaw Workspace

实际项目代码已收拢到 [`pocketclaw/`](./pocketclaw)。

- 项目入口：[`pocketclaw/README.md`](./pocketclaw/README.md)
- 开发目录：`/root/.openclaw/workspace/pocketclaw`

快速开始：

```bash
cd pocketclaw
flutter --version
dart --version
dart pub global activate melos
melos bootstrap
```

仓库根目录保留的是工作区级内容（OpenClaw 本地助手文件、脚本、记录等）以及仓库级基础设施（如 `.github/`、`LICENSE`、`.gitignore`）。
