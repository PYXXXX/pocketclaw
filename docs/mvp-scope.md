# MVP Scope

PocketClaw starts with a chat-first mobile MVP.

## In scope

- connection, authentication, and pairing
- chat history loading
- message sending
- streaming assistant output
- tool event rendering
- session switching
- client-created sessions via `sessionKey`
- image sending
- basic session overrides (`model`, `thinking`, `fast`, `verbose`)

## Out of scope for the first cut

- Gateway-side feature changes
- archive restore workflows
- heavy admin/control surfaces
- browser-wrapper product strategy
