# Assets

This directory stores repository visual assets for PocketClaw.

## Files

- `pocketclaw-banner.svg` — README banner for the repository front page
- `pocketclaw-ui-mockup.svg` — UI direction mockup used in README
- `pocketclaw-logo.svg` — PocketClaw logo / icon concept
- `pocketclaw-social-preview.svg` — source artwork for repository social preview
- `pocketclaw-social-preview.png` — ready-to-upload social preview image for GitHub settings

These assets are lightweight, source-controlled, and safe to render directly from GitHub.
